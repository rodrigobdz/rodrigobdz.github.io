# Website

Code to generate my personal website.

## Requirements

- [`hugo`](https://gohugo.io/)

  ```sh
  brew install hugo
  ```

## Usage

- Start local server

  ```sh
  ./scripts/start-server
  ```

- Update theme

  ```sh
  ./scripts/update
  ```
